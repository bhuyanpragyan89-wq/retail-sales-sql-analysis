# Retail Sales Analysis using SQL

## Overview
This project analyzes retail sales data to understand customer behavior, sales trends, and business performance using SQL.

## Tools Used
- PostgreSQL
- SQL (Joins, Aggregations, Window Functions, CTE)

## Dataset
The dataset contains information about:
- Customers
- Orders
- Sales transactions

## Key Analysis Performed
- Total sales and average order value per customer
- Top customers based on spending
- Sales distribution across cities
- Running total of sales over time
- Customer ranking using window functions
- Identifying customers spending above average
- Customer contribution percentage to total sales
- Top customer in each city

## Key Insights
- A small group of customers contributes a significant portion of total revenue
- Certain cities generate higher overall sales
- Sales show a consistent trend over time
- Top-performing customers can be identified for targeted marketing

## Project Structure
- `retail_sales_analysis.sql` → Contains all SQL queries used in the analysis

## Conclusion
This project demonstrates the use of SQL for data analysis, including aggregations, window functions, and CTEs to derive meaningful business insights.
